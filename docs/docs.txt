The (Browser) class uses Puppeteer’s browser object to manage browser tabs (pages).
The (Page) class uses Puppeteer’s page object to interact with the content of a web page.
The (Element) class uses Puppeteer’s ElementHandle object to interact with DOM elements on a web page.
The (Crawler) class uses the Page and Element classes to navigate through web pages and extract data.

constructor(browser): Initializes a new instance of the Browser class.
async newPage(): Opens a new tab in the browser and returns a Page object.
async close(): Closes the browser.

constructor(selectors): Initializes a new instance of the Crawler class with the given selectors.
addValue(obj): Adds an object to the stored array.
get getStoredAll(): Returns all stored objects.
getStored(key): Returns the stored object at the given key.
async crawl(page, link): Navigates to the given link, scrapes data based on the selectors, adds the data to the stored array, and navigates back.
async parse(page, callback, depth=3, default_val=undefined): Recursively navigates through hrefs on the page and applies the callback function. Stops when depth reaches 0 or when the callback function returns a non-default value.
async saveAsCsv(path): Saves the stored data to a CSV file at the given path.
